# 圖片壓縮

這是一個基於 Vue 3 和 Element Plus 的圖片壓縮應用程式。它允許使用者上傳多張圖片，進行壓縮，並下載壓縮後的圖片。

## 功能

- 拖放或點擊上傳多張圖片
- 預覽上傳的圖片
- 壓縮圖片
- 顯示原始和壓縮後的檔案大小
- 計算壓縮比率
- 下載壓縮後的圖片
- 響應式設計，適合桌上型電腦和行動裝置

## 使用技術

- Vue 3
- Element Plus
- browser-image-compression

## 注意事項

- 本應用程式在客戶端進行圖片壓縮，不會上傳圖片到伺服器。
- 壓縮設定為最大 1MB 大小和 1920px 寬度或高度。
- 支援的圖片格式取決於瀏覽器的支援情況。
